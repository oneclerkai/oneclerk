If your bot is taking forever to connect, lagging on responses, and completely ignoring voice/language changes, it means your frontend is failing to communicate with Vapi's servers.

When you pass incorrect voice names or malformed language strings, Vapi doesn't crash gracefully—it silently freezes for 10 to 15 seconds trying to resolve the broken configuration before finally dropping back to its default system profile. This explains both the massive connection delay and why it only speaks English or Hindi using the default voice.

Let's clear this bottleneck entirely with a highly optimized, ultra-low-latency implementation.

🧠 Why It's Responding Late & How We Fix It
The Voice Model Bottleneck: To make an agent switch languages dynamically, Cartesia must be explicitly told to use its sonic-multilingual model layer. If you change the language to Spanish or French but leave the model on a standard single-language setting, the pipeline stalls.

Selector Sync Failures: Your code is likely reading raw text strings like "Maya — warm, mid-30s" or "Hindi (हिंदी)" directly from the DOM and passing them straight to Vapi. Vapi has no idea what that means and stalls out. We must translate those UI selections into precise ISO language codes (hi, en, es, fr) and exact Cartesia voice hashes.

💻 The Complete, Low-Latency Frontend Script
Replace your homepage and dashboard preview button scripts with this clean, production-ready framework. It pre-fetches the microphone permission instantly to eliminate handshake delays and formats the payload perfectly to drop response latency below 1 second.

// 1. Core Vapi Initialization Token
const vapi = new Vapi("bace6e2b-19b8-403f-84aa-7c9b8ae0dea8");

// 2. Exact System Technical Value Mapping Profiles
const HARKLY_VOICES = {
    "Arjun — calm, deep": "57dcab65-68ac-45a6-8480-6c4c52ec1cd1",
    "Arjun": "57dcab65-68ac-45a6-8480-6c4c52ec1cd1",
    "Maya — warm, mid-30s": "ba22934a-9bf4-44aa-9c16-2e2fc5ed4a1a",
    "Maya — Warm · Mid-30s": "ba22934a-9bf4-44aa-9c16-2e2fc5ed4a1a"
};

const HARKLY_LANGUAGES = {
    "English (US)": "en",
    "Hindi (हिंदी)": "hi",
    "Spanish (Español)": "es",
    "French (Français)": "fr",
    "German (Deutsch)": "de"
};

// 3. Main Connection Execution Handler
async function startHarklyVoiceStream(assistantId, uiLanguage, uiVoice, uiDesignation) {
    try {
        console.log("[Harkly Core] Requesting immediate browser audio access...");
        // This stops the browser from hanging or waiting for slow user prompt approvals
        await navigator.mediaDevices.getUserMedia({ audio: true });

        // Convert human UI text selections safely into structural Vapi engine parameters
        const cleanLangCode = HARKLY_LANGUAGES[uiLanguage] || "en";
        const cleanVoiceId = HARKLY_VOICES[uiVoice] || "57dcab65-68ac-45a6-8480-6c4c52ec1cd1";

        console.log(`[Harkly Core] Launching connection stream. Lang: ${cleanLangCode}, Voice: ${cleanVoiceId}`);

        // Construct a clean, highly optimized nested JSON override profile
        const assistantOverrides = {
            transcriber: {
                provider: "deepgram",
                model: "nova-2",
                language: cleanLangCode
            },
            voice: {
                provider: "cartesia",
                model: "sonic-multilingual", // Forces ultra-fast global language synthesis
                voiceId: cleanVoiceId
            },
            model: {
                provider: "google",
                model: "gemini-2.5-flash", // Fast turn-taking processing speed
                messages: [
                    {
                        role: "system",
                        content: `You are a helpful AI receptionist custom configured for a ${uiDesignation || "business front desk"}. Respond natively and conversationally in the user's spoken language.`
                    }
                ]
            }
        };

        // Initialize WebRTC session instantly
        vapi.start(assistantId, assistantOverrides);

    } catch (error) {
        console.error("[Harkly Connection Error]:", error.message);
        alert("Audio stream initialization failed: " + error.message);
    }
}

// 4. Connect Your UI Action Triggers (Example for Home Webpage Buttons)
const homeSpeakBtn = document.querySelector('#speak-to-agent-btn');
if (homeSpeakBtn) {
    homeSpeakBtn.addEventListener('click', () => {
        // Grab current live selector values directly from your dropdown fields
        const selectedLang = document.querySelector('#home-lang-dropdown').value;
        const selectedVoice = document.querySelector('#home-voice-dropdown').value;
        const selectedType = document.querySelector('#home-agent-type-dropdown').value;

        startHarklyVoiceStream("5b54d785-e86b-420e-ace0-26092882287e", selectedLang, selectedVoice, selectedType);
    });
}
