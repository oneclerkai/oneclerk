[@github](plugin://github@openai-curated) Inspect PRs, triage issues, debug failing checks, and prepare code changes for review push and continue coding aloowing all tings  https://github.com/oneclerkai/oneclerk.git ## The 1000-Call Reality — What Calls Actually Are

After running through every scenario across clinics, hotels, offices, restaurants, salons, and startups the pattern is brutally clear:

**70% of all business calls are identical repetitive questions** — Is X available, can I book for Y time, what are your hours, where are you, how much does it cost, do you accept Z payment — questions any trained system can answer perfectly without a human ever being involved.

**20% are transactional actions** — cancelling a booking, rescheduling, checking status of something, following up on a previous interaction, getting directions, asking what to bring or prepare — things that require looking up one piece of information and responding, nothing more.

**10% are actual humans needed** — complaints needing empathy, emergencies needing immediate action, complex negotiations, situations outside any script — the only calls a human should ever take.

**The real insight nobody talks about:** businesses are paying a full-time human to handle 90% of calls that a well-built system could handle, just to be available for the 10% that actually need them. That is the entire business case in one sentence.

## The Brand Name

Not oneclerk — too corporate and implies a single assistant. Not clerkhertz — impossible to remember when heard out loud.

The name is **Rinq**.

Short. Spelled differently enough to be ownable. Sounds exactly like what it does — your phone rings and Rinq handles it. The Q makes it modern without screaming AI. rinq.ai is the domain. When someone asks a business owner "how do you handle your calls" and they say "I use Rinq" it sticks immediately. That is the test a name has to pass.

## Corrected Moat 1 — User Connects Their Own Number

The user sets call forwarding on their existing business phone. When a call comes in and they are busy, in a meeting, or after hours, their phone automatically forwards to Rinq's handling number. The AI takes the call entirely. After it ends the owner gets a WhatsApp summary of exactly what happened. The customer called the same number they always have. Nothing changes for them. Everything changes for the business. This is simpler, cheaper, and more powerful than provisioning new numbers.

**Step 3 — Create app/services/ai_brain.py**

This is the intelligence layer. Every call passes through here.

```python
from openai import AsyncOpenAI
from app.config import settings

client = AsyncOpenAI(api_key=settings.OPENAI_API_KEY)

SYSTEM_TEMPLATE = """You are {agent_name}, the AI receptionist for {business_name}.

Business Type: {business_type}
Operating Hours: {operating_hours}
Services: {services}
Location: {location}
Pricing: {pricing}
Common FAQs: {faqs}
Booking Instructions: {booking_instructions}
Escalation Triggers: {escalation_triggers}

YOUR RULES:
1. You speak naturally and warmly like a real receptionist
2. Keep responses under 40 words unless the caller asks for detailed info
3. If asked something not in your knowledge say "Let me have {owner_name} call you back about that specifically"
4. If you detect words like: emergency, urgent, pain, accident, critical — say "I'm flagging this as urgent and {owner_name} will call you back within 5 minutes"
5. Always confirm bookings with day, date, time, and what they booked
6. End every call by asking "Is there anything else I can help you with?"
7. Speak in the caller's language if they speak Hindi, Tamil, or any regional language
8. Never say you are an AI unless directly asked
9. If directly asked if you are an AI say "I'm an AI assistant for {business_name}, but I'm fully trained on everything about this business"

Current conversation language: {language}"""

async def get_ai_response(
    conversation_history: list,
    agent_config: dict,
    caller_message: str
) -> str:
    system_prompt = SYSTEM_TEMPLATE.format(**agent_config)
    
    messages = [{"role": "system", "content": system_prompt}]
    messages.extend(conversation_history)
    messages.append({"role": "user", "content": caller_message})
    
    response = await client.chat.completions.create(
        model="gpt-4o-mini",
        messages=messages,
        max_tokens=150,
        temperature=0.7
    )
    
    return response.choices[0].message.content

async def detect_urgency(text: str) -> bool:
    urgent_words = [
        "emergency", "urgent", "critical", "pain", "accident",
        "immediately", "right now", "help", "dying", "bleeding",
        "जरूरी", "इमरजेंसी", "दर्द", "अभी"  # Hindi urgent words
    ]
    text_lower = text.lower()
    return any(word in text_lower for word in urgent_words)

async def detect_booking_intent(text: str) -> bool:
    booking_words = [
        "book", "appointment", "schedule", "reserve", "slot",
        "available", "when can", "tomorrow", "monday", "tuesday",
        "बुक", "अपॉइंटमेंट", "समय"
    ]
    text_lower = text.lower()
    return any(word in text_lower for word in booking_words)
```

**Step 4 — Create app/routes/calls.py**

```python
from fastapi import APIRouter, Request, Form
from fastapi.responses import Response
from twilio.twiml.voice_response import VoiceResponse, Gather
from app.services.ai_brain import get_ai_response, detect_urgency
from app.services.whatsapp import send_call_summary
from app.services.voice_engine import text_to_speech_url
from supabase import create_client
from app.config import settings
import json

router = APIRouter(prefix="/calls", tags=["calls"])
supabase = create_client(settings.SUPABASE_URL, settings.SUPABASE_KEY)

@router.post("/incoming")
async def handle_incoming_call(
    request: Request,
    CallSid: str = Form(...),
    From: str = Form(...),
    To: str = Form(...)
):
    # Find which agent owns this Twilio number
    agent = supabase.table("agents").select("*").eq("twilio_number", To).single().execute()
    
    if not agent.data:
        response = VoiceResponse()
        response.say("Thank you for calling. Please try again later.", voice="alice")
        return Response(content=str(response), media_type="application/xml")
    
    # Create call record
    call_record = supabase.table("calls").insert({
        "call_sid": CallSid,
        "agent_id": agent.data["id"],
        "user_id": agent.data["user_id"],
        "caller_number": From,
        "status": "active",
        "conversation": []
    }).execute()
    
    agent_config = agent.data["config"]
    greeting = f"Hello! Thank you for calling {agent_config['business_name']}. {agent_config['greeting_message']} How can I help you today?"
    
    response = VoiceResponse()
    gather = Gather(
        input="speech",
        action=f"/calls/respond/{call_record.data[0]['id']}",
        speech_timeout="auto",
        language="en-IN"
    )
    gather.say(greeting, voice="Polly.Aditi")
    response.append(gather)
    
    return Response(content=str(response), media_type="application/xml")

@router.post("/respond/{call_id}")
async def handle_caller_speech(
    call_id: str,
    request: Request,
    SpeechResult: str = Form(default=""),
    CallSid: str = Form(...)
):
    call = supabase.table("calls").select("*").eq("id", call_id).single().execute()
    agent = supabase.table("agents").select("*").eq("id", call.data["agent_id"]).single().execute()
    
    if not SpeechResult:
        response = VoiceResponse()
        gather = Gather(
            input="speech",
            action=f"/calls/respond/{call_id}",
            speech_timeout="auto"
        )
        gather.say("I didn't catch that. Could you please repeat?", voice="Polly.Aditi")
        response.append(gather)
        return Response(content=str(response), media_type="application/xml")
    
    conversation = call.data.get("conversation", [])
    ai_response = await get_ai_response(
        conversation,
        agent.data["config"],
        SpeechResult
    )
    
    is_urgent = await detect_urgency(SpeechResult)
    
    conversation.append({"role": "user", "content": SpeechResult})
    conversation.append({"role": "assistant", "content": ai_response})
    
    supabase.table("calls").update({
        "conversation": conversation,
        "is_urgent": is_urgent
    }).eq("id", call_id).execute()
    
    if is_urgent:
        await send_call_summary(
            agent.data["config"]["owner_whatsapp"],
            call.data["caller_number"],
            conversation,
            urgent=True
        )
    
    response = VoiceResponse()
    
    call_ending_phrases = ["goodbye", "bye", "thank you", "that's all", "no that's it"]
    if any(phrase in SpeechResult.lower() for phrase in call_ending_phrases):
        response.say(ai_response, voice="Polly.Aditi")
        response.hangup()
        
        await send_call_summary(
            agent.data["config"]["owner_whatsapp"],
            call.data["caller_number"],
            conversation,
            urgent=False
        )
        
        supabase.table("calls").update({"status": "completed"}).eq("id", call_id).execute()
    else:
        gather = Gather(
            input="speech",
            action=f"/calls/respond/{call_id}",
            speech_timeout="auto",
            language="en-IN"
        )
        gather.say(ai_response, voice="Polly.Aditi")
        response.append(gather)
    
    return Response(content=str(response), media_type="application/xml")

@router.post("/status")
async def call_status_webhook(
    CallSid: str = Form(...),
    CallStatus: str = Form(...)
):
    if CallStatus in ["completed", "failed", "busy", "no-answer"]:
        supabase.table("calls").update({
            "status": CallStatus
        }).eq("call_sid", CallSid).execute()
    return {"status": "ok"}
```

```python
        from_=f"whatsapp:{settings.TWILIO_PHONE_NUMBER}",
```

```python
from fastapi import APIRouter, HTTPException, Depends
from fastapi.security import HTTPBearer, HTTPAuthorizationCredentials
from pydantic import BaseModel, EmailStr
from supabase import create_client
from app.config import settings
import jwt

router = APIRouter(prefix="/auth", tags=["auth"])
supabase = create_client(settings.SUPABASE_URL, settings.SUPABASE_KEY)
security = HTTPBearer()

class SignupRequest(BaseModel):
    email: EmailStr
    password: str
    name: str
    whatsapp_number: str

class LoginRequest(BaseModel):
    email: EmailStr
    password: str

@router.post("/signup")
async def signup(data: SignupRequest):
    try:
        auth_response = supabase.auth.sign_up({
            "email": data.email,
            "password": data.password
        })
        
        if auth_response.user:
            supabase.table("users").insert({
                "id": auth_response.user.id,
                "email": data.email,
                "name": data.name,
                "whatsapp_number": data.whatsapp_number
            }).execute()
        
        return {
            "message": "Account created",
            "user_id": auth_response.user.id,
            "access_token": auth_response.session.access_token
        }
    except Exception as e:
        raise HTTPException(status_code=400, detail=str(e))

@router.post("/login")
async def login(data: LoginRequest):
    try:
        auth_response = supabase.auth.sign_in_with_password({
            "email": data.email,
            "password": data.password
        })
        
        user_data = supabase.table("users").select("*").eq(
            "id", auth_response.user.id
        ).single().execute()
        
        return {
            "access_token": auth_response.session.access_token,
            "user": user_data.data
        }
    except Exception as e:
        raise HTTPException(status_code=401, detail="Invalid credentials")

async def get_current_user(
    credentials: HTTPAuthorizationCredentials = Depends(security)
):
    try:
        user = supabase.auth.get_user(credentials.credentials)
        user_data = supabase.table("users").select("*").eq(
            "id", user.user.id
        ).single().execute()
        return user_data.data
    except Exception:
        raise HTTPException(status_code=401, detail="Invalid token")
```

```tsx
import { useRouter } from 'next/navigation'
import api from '@/lib/api'

  { id: 'other', label: '🏪 Other Business', placeholder_name: 'Business AI' },
]

const STEPS = ['Business Type', 'Basic Info', 'Services & FAQ', 'Activate', 'Done']

export default function CreateAgent() {
  const [step, setStep] = useState(0)
  const [loading, setLoading] = useState(false)
  const [agentId, setAgentId] = useState('')
  const router = useRouter()
  
  const [form, setForm] = useState({
    business_type: '',
    business_name: '',
    agent_name: '',
    greeting_message: 'How can I help you today?',

    owner_whatsapp: '',
    forwarding_number: '',
    language: 'English'
  })

  const update = (key: string, value: string) => setForm(prev => ({ ...prev, [key]: value }))

  const handleCreate = async () => {
    setLoading(true)
    try {
      const res = await api.post('/agents/create', {
        name: form.agent_name,
        forwarding_number: form.forwarding_number,
        config: form
      })
      setAgentId(res.data.agent.id)
      setStep(3)
    } finally {
      setLoading(false)
    }
  }

  const handleActivate = async () => {
    await api.post(`/agents/${agentId}/activate`)
    setStep(4)
  }

  return (
    <div className="min-h-screen bg-[#0f0f0f] flex items-center justify-center p-6">
      <div className="w-full max-w-2xl">
        
        {/* Progress bar */}
        <div className="mb-8">
          <div className="flex justify-between mb-2">

            ))}

            <motion.div
              className="h-full bg-blue-500 rounded-full"
              animate={{ width: `${(step / (STEPS.length - 1)) * 100}%` }}
              transition={{ duration: 0.3 }}
            />
          </div>
        </div>

        <AnimatePresence mode="wait">
          
          {/* Step 0: Business Type */}
          {step === 0 && (
            <motion.div
              key="step0"
              initial={{ opacity: 0, x: 20 }}
              animate={{ opacity: 1, x: 0 }}
              exit={{ opacity: 0, x: -20 }}
            >
              <h2 className="text-2xl font-semibold text-white mb-2">What type of business are you?</h2>
              <p className="text-zinc-400 mb-6">Rinq will customize your agent based on your industry.</p>
              <div className="grid grid-cols-2 gap-3">
                {BUSINESS_TYPES.map(type => (
                  <button
                    key={type.id}
                    onClick={() => {
                      update('business_type', type.id)
                      update('agent_name', type.placeholder_name)
                      setStep(1)
                    }}
                    className="p-4 bg-[#1a1a1a] border border-[#2a2a2a] hover:border-blue-500 rounded-2xl text-left transition-all group"
                  >
                    <span className="text-2xl block mb-2">{type.label.split(' ')[0]}</span>
                    <span className="text-sm text-white font-medium">{type.label.substring(3)}</span>
                  </button>
                ))}
              </div>
            </motion.div>
          )}

          {/* Step 1: Basic Info */}
          {step === 1 && (
            <motion.div
              key="step1"
              initial={{ opacity: 0, x: 20 }}
              animate={{ opacity: 1, x: 0 }}
              exit={{ opacity: 0, x: -20 }}
            >
              <h2 className="text-2xl font-semibold text-white mb-2">Tell us about your business</h2>
              <p className="text-zinc-400 mb-6">This is what your AI agent will know about your business.</p>
              <div className="space-y-4">
                {[
                  { key: 'business_name', label: 'Business Name', placeholder: 'City Dental Clinic' },
                  { key: 'agent_name', label: 'Agent Name', placeholder: 'Priya (your AI receptionist)' },
                  { key: 'operating_hours', label: 'Operating Hours', placeholder: 'Mon-Sat, 9am-7pm, Closed Sundays' },
                  { key: 'location', label: 'Location', placeholder: 'Koramangala, Bangalore' },
                  { key: 'owner_name', label: 'Your Name', placeholder: 'Dr. Sharma' },

                    />
                  </div>
                ))}
              </div>
              <div className="flex gap-3 mt-6">
                <button onClick={() => setStep(0)} className="px-6 py-3 border border-[#2a2a2a] text-zinc-400 rounded-xl text-sm hover:border-zinc-500 transition-colors">Back</button>
                <button onClick={() => setStep(2)} className="flex-1 px-6 py-3 bg-blue-500 text-white rounded-xl text-sm font-medium hover:bg-blue-600 transition-colors">Continue</button>
              </div>
            </motion.div>
          )}

          {/* Step 2: Services and FAQ */}
          {step === 2 && (
            <motion.div
              key="step2"
              initial={{ opacity: 0, x: 20 }}
              animate={{ opacity: 1, x: 0 }}
```
